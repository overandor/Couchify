# Development History

This folder records Couchify development progress as public-safe Markdown notes.

Each entry should include:

- date and local time;
- system being developed;
- decision made;
- files changed;
- current status;
- next concrete build step.

## Current direction

Couchify is being documented as a consent-first AI seat and local artifact network. The development history folder will preserve how the idea evolves from manifesto to API specification, Swagger documentation, demo, and implementation.
